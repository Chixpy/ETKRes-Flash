Shady Room Escape is a type of point and click new escape game developed by games2rule.com. You are trapped inside a shady room. In this shady room you can't see any object properly. You don't know where is key of door and how to escape from there. Now you should use you escaping skill. Search useful object and find the way of escape from that shady room. Explore the shady room and get necessary items and solve all puzzles to escape from this place. Good Luck and Have fun!
