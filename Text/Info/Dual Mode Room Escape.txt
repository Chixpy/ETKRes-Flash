Dual Mode Room Escape is type of point and click new escape game developed by games2rule.com. You are trapped inside in a Dual Mode Room. The door of the Dual Mode room is locked. There is no one near to help you out. This dual mode room will change its appearance in every 30 seconds. We cannot able to take some objects in mode but we can take same objects in other mode. Find some useful objects and hints to escape from the Dual Mode Room. Good Luck and Have Fun!
