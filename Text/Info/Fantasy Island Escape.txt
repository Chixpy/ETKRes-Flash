Fantasy Island Escape is another type of point and click new escape game developed by wowescape.com. In this escape game you are trapped inside a fantasy island. This island is too beautiful between the seas but except you no one is there. You don't know way of escape from that fantasy island and how you have to escape from there. You have been searching for the exit path of Fantasy Island where you have been trapped. Try to find some clues and objects which make an escape path for you. Click on the objects to interact with them and solve fantasy puzzles. Good Luck. Have a fun.
