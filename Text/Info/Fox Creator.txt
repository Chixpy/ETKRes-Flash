How to get all the Achievements to unlock all the options:
    * Neck Accessories: Open the Game for the first time.
    * Fox Masks: Read the in-game instructions until the end.
    * Scars: Use Ctrl+Z or Reset several times.
    * Decoration: Use the Random button several times.
    * Head accessories: Open the game for the second time.
    * Leg Accessories: Open the game for the third or fourth time.
    * Fox Fire: Use the game the next day.
    * Wings: Unlocked after you have obtained all the other Achievements.
