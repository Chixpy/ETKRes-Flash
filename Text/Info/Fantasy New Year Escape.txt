Fantasy New Year esacape is a type of live escape game developed by Wowescape.com. In this escape game you are trapped inside a fantasy forest place. This place is too beautiful but except you no one is there. In this place there is a fantasy calendar which is stuck on 26 December. If it will be move it on 1st Jan then you can escape from this place. Otherwise you will be permanently trapped there. So you have to move the calendar day by day. This the game of fantasy New Year escape in this game you have to move the calendar date 26 to 1st Jan by finding the clues, objects and solving puzzle which is hidden anywhere in that fantasy place. So let's start use your escaping mind and escape from that fantasy place. Good Luck. Have a fun.
